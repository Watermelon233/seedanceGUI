## 📋 功能概述

首尾帧参考模式是Seedance GUI的高级功能，允许用户指定视频的第一帧和最后一帧，AI会自动生成从首帧到尾帧的过渡动画。

### 🔥 核心特性

1. **精准控制**: 指定视频的起始和结束状态
2. **自适应比例**: 支持`adaptive`宽高比，根据图片自动调整
3. **智能过渡**: AI自动生成流畅的过渡动画
4. **360度运镜**: 支持环绕运镜等复杂效果

## 🚀 API调用示例

### 标准首尾帧调用

```json
{
  "model": "doubao-seedance-1-0-pro-250528",
  "content": [
    {
      "type": "text",
      "text": "360度环绕运镜"
    },
    {
      "type": "image_url",
      "image_url": {
        "url": "https://example.com/first_frame.jpg"
      },
      "role": "first_frame"
    },
    {
      "type": "image_url",
      "image_url": {
        "url": "https://example.com/last_frame.jpg"
      },
      "role": "last_frame"
    }
  ],
  "duration": 5,
  "ratio": "adaptive"
}
```

### 关键参数说明

- **first_frame**: 视频的第一帧（起始状态）
- **last_frame**: 视频的最后一帧（结束状态）
- **ratio**: 使用`"adaptive"`让系统根据图片自动调整宽高比
- **model**: 推荐使用`doubao-seedance-1-0-pro-250528`专业版模型

## ⚠️ 使用注意事项

### 1. 图片数量要求
- **首帧参考**: 必须**恰好2张图片**
- **尾帧参考**: 必须**恰好2张图片**
- 第1张 = first_frame（起始状态）
- 第2张 = last_frame（结束状态）

### 2. 宽高比选择
- **推荐使用**: `adaptive`（自适应）
- 系统会根据首尾帧图片自动调整视频比例
- 避免图片变形或裁剪

### 3. 模型选择
- **最佳效果**: `doubao-seedance-1-0-pro-250528`
- **平衡性能**: `doubao-seedance-1-0-lite-i2v-250428`
- 其他Seedance模型也支持首尾帧模式

### 4. 提示词建议
- 简洁描述运镜效果：`"360度环绕运镜"`
- 描述过渡方式：`"平滑过渡从白天到黑夜"`
- 指定相机运动：`"缓慢推近，从广角到特写"`

## 📈 功能对比

### 参考模式对比

| 模式 | 图片数量 | Role设置 | 适用场景 |
|------|----------|----------|----------|
| **全能参考** | 1-9张 | 全部为`reference_image` | 综合参考多张图片风格 |
| **首帧参考** | 恰好2张 | 第1张`first_frame`，第2张`last_frame` | 指定起始和结束状态 |
| **尾帧参考** | 恰好2张 | 第1张`first_frame`，第2张`last_frame` | 指定起始和结束状态 |

### Ratio选择对比

| 选项 | 适用场景 | 特点 |
|------|----------|------|
| **adaptive** | 首尾帧模式 | 根据图片自动调整比例，避免变形 |
| **16:9** | 横屏视频 | 标准横屏比例 |
| **9:16** | 竖屏视频 | 适合手机竖屏观看 |
| **1:1** | 方形视频 | 适合社交媒体发布 |
