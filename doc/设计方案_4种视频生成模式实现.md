## 项目概述

### 背景

当前 Seedance 2.0 应用支持3种参考模式（全能参考、首帧参考、尾帧参考），但根据官方API文档，实际存在4种视频生成模式：

1. **文生视频** - 纯文本生成
2. **图生视频-首帧** - 单张图片作为首帧
3. **图生视频-首尾帧** - 两张图片控制首尾
4. **图生视频-参考图** - 多张图片风格参考

### 目标

1. ✅ 在 `ApiTestPage` 和 `SingleTaskPage` 添加4种模式选择
2. ✅ 修正当前代码中的 role 逻辑错误
3. ✅ 实现类型安全的模式管理系统
4. ✅ 保持有声/无声作为独立配置项

### 范围

- **包含**: ApiTestPage.tsx、SingleTaskPage.tsx、types/index.ts
- **不包含**: 后端API修改（优先使用前端映射方案）

#### 1.3 辅助函数

```typescript
/**
 * 获取模式配置
 */
export function getModeConfig(mode: VideoGenerationMode): ModeConfig {
  return MODE_CONFIG[mode];
}

/**
 * 验证图片数量
 */
export function validateImageCount(mode: VideoGenerationMode, count: number): {
  valid: boolean;
  error?: string;
} {
  const config = getModeConfig(mode);

  if (count < config.minImages) {
    return {
      valid: false,
      error: `${MODE_OPTIONS.find(m => m.value === mode)?.label}模式至少需要${config.minImages}张图片`
    };
  }

  if (count > config.maxImages) {
    return {
      valid: false,
      error: `${MODE_OPTIONS.find(m => m.value === mode)?.label}模式最多支持${config.maxImages}张图片`
    };
  }

  return { valid: true };
}

/**
 * 根据模式获取图片的role
 */
export function getImageRole(mode: VideoGenerationMode, index: number): string | undefined {
  const config = getModeConfig(mode);

  if (!config.useRole) {
    return undefined;
  }

  if (config.roleType === 'first_last') {
    return index === 0 ? 'first_frame' : 'last_frame';
  }

  if (config.roleType === 'reference') {
    return 'reference_image';
  }

  return undefined;
}
```

#### 2.2 模式切换逻辑

```typescript
const handleModeChange = (newMode: VideoGenerationMode) => {
  setSelectedMode(newMode);

  const config = getModeConfig(newMode);

  // 自动调整ratio
  setTestParams(prev => ({
    ...prev,
    ratio: config.defaultRatio
  }));

  // 清空或截断图片
  if (config.maxImages === 0) {
    setImageFiles([]);
  } else if (imageFiles.length > config.maxImages) {
    setImageFiles(prev => prev.slice(0, config.maxImages));
  }

  // 清空验证错误和请求体
  setValidationErrors([]);
  setGeneratedRequestBody(null);
};
```

#### 2.3 请求体构建（核心修正）

```typescript
const buildRequestBody = async () => {
  const errors = validateParams();
  setValidationErrors(errors);

  if (errors.length > 0) return;

  try {
    const config = getModeConfig(selectedMode);
    const content: any[] = [];

    // 添加文本
    content.push({
      type: 'text',
      text: testParams.prompt.trim()
    });

    // 添加图片（修正role逻辑）
    if (imageFiles.length > 0) {
      for (let i = 0; i < imageFiles.length; i++) {
        const file = imageFiles[i];
        const base64 = await fileToBase64(file);

        const imageItem: any = {
          type: 'image_url',
          image_url: {
            url: base64
          }
        };

        // ⚠️ 关键修正：根据配置决定是否添加role
        if (config.useRole) {
          const role = getImageRole(selectedMode, i);
          if (role) {
            imageItem.role = role;
          }
        }

        content.push(imageItem);
      }
    }

    // 构建请求体
    const requestBody = {
      model: MODEL_ID_MAPPING[testParams.model],
      content: content,
      resolution: testParams.model.includes('vip') ? '720p' : testParams.resolution,
      ratio: testParams.ratio,
      duration: testParams.duration,
      generate_audio: testParams.generateAudio,
      seed: testParams.seed,
      watermark: testParams.watermark,
      camera_fixed: false
    };

    setGeneratedRequestBody(requestBody);
  } catch (error) {
    setValidationErrors([`❌ 构建请求体失败: ${error}`]);
  }
};
```

```typescript
function convertToBackendMode(mode: VideoGenerationMode): {
  referenceMode: string;
  useRole: boolean;
} {
  switch (mode) {
    case VideoGenerationMode.TEXT_TO_VIDEO:
      return { referenceMode: '全能参考', useRole: false };
    case VideoGenerationMode.IMAGE_TO_VIDEO_FIRST_FRAME:
      return { referenceMode: '首帧参考', useRole: false };
    case VideoGenerationMode.IMAGE_TO_VIDEO_FIRST_LAST:
      return { referenceMode: '首帧参考', useRole: true };
    case VideoGenerationMode.IMAGE_TO_VIDEO_REFERENCE:
      return { referenceMode: '全能参考', useRole: true };
  }
}
```

#### B.2 图片role生成

```typescript
function generateImageContent(
  mode: VideoGenerationMode,
  images: File[]
): Promise<any[]> {
  const config = getModeConfig(mode);
  const content: any[] = [];

  for (let i = 0; i < images.length; i++) {
    const base64 = await fileToBase64(images[i]);
    const imageItem: any = {
      type: 'image_url',
      image_url: { url: base64 }
    };

    // 根据配置决定是否添加role
    if (config.useRole) {
      const role = getImageRole(mode, i);
      if (role) {
        imageItem.role = role;
      }
    }

    content.push(imageItem);
  }

  return content;
}
```
