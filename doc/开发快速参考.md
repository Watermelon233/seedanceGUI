## 核心概念速查

### 双会话模型

| 会话类型 | 存储位置 | 用途 |
|----------|----------|------|
| **平台登录会话** | `localStorage: seedance_session_id` | 应用鉴权 |
| **即梦 Session** | `jimeng_session_accounts` 表 | 调用即梦 API |

### 任务模型

```
draft (草稿)
  ├─ prompt: 提示词
  ├─ assets: 图片素材
  └─ video_count: 计划输出数
       ↓ 展开
output (生成任务) × N
  ├─ source_task_id: 来源 draft
  ├─ output_index: 输出序号
  └─ status: pending/generating/done/error
```

### Session 解析优先级

```
user_default
  ↓ (无)
legacy_global (settings.session_id)
  ↓ (无)
env_default (VITE_DEFAULT_SESSION_ID)
  ↓ (无)
none (未配置)
```

### 权限规则

| 角色 | 可见范围 |
|------|----------|
| 普通用户 | 自己的项目/任务/批量/下载 |
| 管理员 | 全部数据 |

## API 请求模式

### 需要登录态的接口

前端调用时需显式添加 `getAuthHeaders()`：

```typescript
// 项目/任务/批量/下载/Session 账号/视频生成
import { getAuthHeaders } from './authService';

fetch('/api/projects', {
  headers: getAuthHeaders()
});
```

### 不需要显式认证头的接口

```typescript
// 当前前端未显式附加
getSettings()          // /api/settings
pollTaskStatus()       // /api/task/:taskId
```

## 调试技巧

### 查看生效 Session

```javascript
// 后端日志
console.log('生效 Session:', effectiveSession);
```

### 查看 ownership 校验

```javascript
// 检查项目归属
SELECT * FROM projects WHERE user_id = ?;
```

### 查看任务展开

```javascript
// draft 展开
SELECT * FROM tasks WHERE source_task_id = ?;
```

## 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 401 Unauthorized | Session 失效 | 重新登录 |
| "未配置可用 SessionID" | 无有效账号 | 设置页添加 |
| "项目不存在" | ownership 校验失败 | 检查 user_id |
| "只能展开 draft 任务" | task_kind 错误 | 检查任务类型 |
