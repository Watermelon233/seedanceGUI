#### 阶段 1: 修复视频下载按钮
- **新建**: `src/services/videoDownloadService.ts`
  - 使用 fetch + blob 方式下载视频
  - 支持从 Content-Disposition 头提取文件名
  - 5分钟超时保护
- **修改**: `src/components/VideoPlayer.tsx`
  - 下载按钮改为调用 `downloadVideoFromProxy()` 函数
  - 支持错误提示

#### 阶段 2: 任务列表管理功能 (修复并发问题)

##### IndexedDB 升级 (v2 -> v3)
- **文件**: `src/services/indexedDBService.ts`
- **新增字段**: `apiKeyHash`, `provider`, `updatedAt`, `lastPolledAt`, `requestPayloadSummary`
- **新增索引**: `apiKeyHash`, `provider`, `lastPolledAt`, `apiKeyHash_status`, `apiKeyHash_createdAt`
- **迁移方式**: 累积式 `oldVersion < N`

##### Provider 接口支持 AbortSignal
- **修改**: 所有 Provider 的 `getTaskStatus()` 方法添加可选 `signal?: AbortSignal` 参数
- **文件**:

  - `src/services/providers/volcengineProvider.ts`
  - `src/services/providers/mockProvider.ts`

##### 共享工具函数
- **新建**: `src/services/providers/utils.ts`
- **函数**: `isAbortError()` - 统一的 AbortError 识别

##### 任务列表服务 (v8 完整版)

## 💾 重要配置信息

### 默认配置
- **默认供应商**: `aihubmix`
- **默认端点**: `https://aihubmix.com`
- **代理服务器**: `http://localhost:3002`
- **Mock 模式**: 通过 `VITE_MOCK_API=true` 启用

### 启动命令
```bash
# 启动代理服务器（必须！）
node proxy-server.cjs

# 启动前端
npm run dev
```
